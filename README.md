📚 Attendly Pro

A modern Student Management Platform built with Flutter, Node.js, Express.js, and MongoDB Atlas to simplify academic management. Attendly Pro helps students track attendance, manage assignments, organize study plans, monitor CGPA, and stay updated with academic events through a single application.

🚀 Features

📊 Dashboard

- Academic overview
- Attendance summary
- Quick access to all modules
- Daily schedule

✅ Attendance Tracker

- Subject-wise attendance management
- Automatic attendance percentage calculation
- Attendance recovery planning
- Safe bunk prediction

📅 Timetable Viewer

- Weekly timetable
- Subject timings
- Classroom details

📖 Academic Calendar

- Upload academic calendar PDF
- Semester schedule
- Holiday tracking
- Important academic dates

📝 Assignment Manager

- Create assignments
- Set priorities
- Due date reminders
- Track completed and pending tasks

📚 Study Planner

- Daily study plans
- Weekly goals
- Monthly study schedule
- Productivity tracking

🎓 CGPA Tracker

- Current CGPA calculation
- Target CGPA setting
- GPA prediction
- Semester-wise progress

📈 Analytics Dashboard

- Attendance analytics
- Assignment progress
- Study progress
- CGPA insights

📢 Campus Notices

- College announcements
- Department notices
- Event updates

📝 Exam Alerts

- Upcoming exams
- Countdown reminders
- Better exam planning

👤 Student Profile

- Personal information
- Academic information
- Attendance summary
